The same editor is available online, as a single portable HTML file, or as an Android application with native file handling.

◎

1. Launch Online

Use Sciwrix immediately in a desktop or mobile browser.

  • Nothing to install
  • Works on Windows, macOS, Linux, Android and modern mobile browsers
  • Access it from any device with an internet connection
  • The editor runs locally in your browser

Getting started

Simple to Use, Easy to Keep

Choose the format that suits your device and how you prefer to manage files.

Using the standalone HTML app

The downloaded file is the complete application. It does not need to be installed.

  1. Download Sciwrix.html.
  2. Open it with a modern browser.
  3. Use Open and Save As to work with Markdown files.
  4. Keep the HTML file anywhere convenient, including offline storage.

Using the Android APK

The APK adds Android-specific file handling while retaining the same editor.

  1. Download the latest APK from the GitHub release.
  2. Allow installation from your browser or file manager when Android asks.
  3. Choose Sciwrix from Open with for Markdown files.
  4. Use Save to update writable files, or Save As to create another copy.
